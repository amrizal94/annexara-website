Keunggulan

Fitur Unggulan
Annexara DronePilot

Setiap fitur dirancang untuk memenuhi kebutuhan nyata petani Indonesia di lapangan.

Navigasi GPS Presisi

Sistem GPS RTK dengan akurasi sentimeter memastikan drone terbang tepat di jalur yang direncanakan, meminimalkan tumpang tindih semprot.

Mission Planning Otomatis

Gambar area lahan di peta, tentukan parameter semprot, dan drone akan menerbangkan misi secara fully autonomous tanpa input manual.

Telemetri Real-Time

Monitor baterai, kecepatan, ketinggian, status pompa, dan sisa cairan secara langsung di layar remote Skydroid H12.

Cloud Logging Firebase

Data setiap penerbangan tersimpan otomatis ke Firebase Cloud. Akses laporan luas area, konsumsi cairan, dan riwayat misi kapan saja.

Sistem Keselamatan Berlapis

Return-to-home otomatis saat baterai kritis, geofencing area terbang, dan obstacle detection untuk keamanan operasional optimal.

Remote Skydroid H12

Kendali presisi via remote controller Skydroid H12 dengan layar built-in, koneksi ke software Annexara melalui serial port langsung.

Cara Kerja

Dari Lahan ke Laporan
dalam 4 Langkah

1

Rencanakan Misi

Buka aplikasi Annexara di remote H12. Gambar batas area lahan di peta, atur ketinggian terbang, kecepatan, dan overlap semprot. Sistem otomatis menghitung jalur optimal dan estimasi waktu.

2

Isi Tangki & Armingkan Drone

Isi tangki 30L dengan pestisida atau pupuk cair sesuai dosis. Pastikan baterai terisi penuh. Drone melakukan pre-flight check otomatis — GPS lock, motor, pompa, dan koneksi telemetri.

3

Terbang & Semprot Otomatis

Tekan Start Mission. Drone terbang dan menyemprot secara autonomous sesuai jalur yang direncanakan. Monitor progress real-time di layar — posisi GPS, volume cairan sisa, luas area tersemprot.

4

Laporan Tersimpan Otomatis

Setelah misi selesai, seluruh data penerbangan (rute, waktu, volume cairan, area tercakup) tersimpan ke Firebase Cloud. Akses kapan saja dari aplikasi atau dashboard web untuk laporan dan analisis.
